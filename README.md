# Voix
Voicemeeter Potato web interface.
For Voicemeeter Banana check out the repo i forked from: https://github.com/karlkoorna/voix

## Usage

Install dependencies.
```
$ npm install
$ npm run build
```

Start for any environment.
```
$ npm start
```

## Limitations

* No Equalizer view.
* No Settings view.
* No VBAN view.
